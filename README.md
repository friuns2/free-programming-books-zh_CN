## What This Repository Is

This repository tracks free programming books and long-form learning resources in Chinese.

The original and complete catalog is preserved in [`README.zh-CN.md`](README.zh-CN.md).
This `README.md` is the English entry point with navigation, contribution workflow, and maintenance status.

## Category Map

Language-agnostic topics include:
- Operating Systems
- Distributed Systems
- Compiler Theory
- Functional Concepts
- Computer Graphics
- Web Servers
- Version Control
- Databases (NoSQL, PostgreSQL, MySQL)
- Design Patterns
- Web and Big Data

Language-specific topics include:
- Android, iOS, App
- C/C++, Clojure, CSS/HTML, Dart, Erlang, Fortran
- Go, Groovy, Haskell, Java, JavaScript
- LaTeX, Lisp, Lua, OCaml, Perl, PHP, Prolog
- Python, R, Ruby, Rust, Scala, Shell, Swift
- Testing and Reading Notes

## Contributing

1. Check for duplicates before adding a resource.
2. Keep entries concise: title + stable link.
3. Submit pull requests against `main`.
4. Use issues for broken links and category requests.

Reference: [`CONTRIBUTING.md`](CONTRIBUTING.md)

## Maintenance Posture

- This fork is actively used for PR triage and backlog consolidation.
- Large catalog changes should keep anchor compatibility when possible.
- Link rot remains the main operational risk.

See [`PROJECT_SPEC.md`](PROJECT_SPEC.md) for current maintenance detail.
