Free Chinese programming book index
============================

[![Build Status](https://travis-ci.org/justjavac/free-programming-books-zh_CN.svg?branch=master)](https://travis-ci.org/justjavac/free-programming-books-zh_CN)

Free index of Chinese programming books, contributions welcome.

- Must-read books for programmers recommended by foreign programmers in [stackoverflow](http://stackoverflow.com/questions/1711/what-is-the-single-most-influential-book-every-programmer-should-read/1713%231713), [Chinese version](http://justjavac.com/other/2012/05/15/qualified-programmer-should-read-what-books.html "What books should a qualified programmer read").
- What are some non-programming books that programmers on [stackoverflow](http://stackoverflow.com/questions/38210/what-non-programming-books-should-programmers-read) should read? [Chinese version](what-non-programming-books-should-programmers-read.md)
- A popular programming book index [Chinese version](https://github.com/vhf/free-programming-books/blob/master/free-programming-books-zh.md:worried:) on [github](https://github.com/vhf/free-programming-books)

Thanks to [@siberiawolf](https://github.com/siberiawolf) for developing the web version using Bootstrap, address: http://siberiawolf.com/free_programming/index.html

## Participate in communication

Everyone is welcome to share the classic free books that have been collected for a long time. You can:

* Use [Issues](https://github.com/justjavac/free-programming-books-zh_CN/issues) and Pull Request

Contributor list: https://github.com/justjavac/free-programming-books-zh_CN/graphs/contributors
